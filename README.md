## Batch-Scripting
Microsoft Windows:
1. Download the Release
2. Unzip the file
3. Run "calc.bat"

Linux:

1. Download wine from https://www.winehq.org/
2. Download the Release
3. Unzip the file&
4. Open a terminal where the file is loctated
5. In the Terminal type ``` wine start calc.bat ```
